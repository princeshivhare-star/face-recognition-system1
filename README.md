# AI Face Attendance System
An automated attendance system using Python, OpenCV, and Firebase.

## Features
- Real-time face detection and recognition.
- Live database updates using Firebase Realtime Database.
- Custom UI with "Loading" and "Success" states.

## Installation
1. Clone the repo.
2. Install requirements: `pip install -r requirements.txt`
3. Add your own `serviceAccountKey.json`.
4. Run `main.py`.
